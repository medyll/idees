# 💡 Incubateur d'Idées

Géré par le système **MATURATION** — les idées grandissent ici, elles ne se développent pas.

---

## Comment ça marche

Chaque idée a sa propre structure :
- `[code-lowercase]/idea.md` — l'état courant, vivant, de l'idée
- `[code-lowercase]/history.md` — le journal horodaté de son évolution (timestamps au format YYYY-MM-DD)

Pour faire évoluer une idée, dire par exemple :
> *"améliore APP-MOBILE-GRID"*, *"ajoute une question à SKILL-MEMOIRE"*, *"fusionne X dans Y"*

## Statuts possibles

| Icône | Statut | Signification |
|-------|--------|--------------|
| 🌱 | Germination | Idée brute, questions ouvertes, peu développée |
| 🌿 | En croissance | L'idée se précise, se développe activement |
| 🌳 | Mature | L'idée est suffisamment claire pour être implémentée |
| 🔗 | Absorbée | Intégrée dans une autre idée |
| 🗃️ | Archivée | Mise de côté, pas abandonnée |
