## Description
Application de test de personnalité. L'idée centrale : la personnalité n'est pas un état figé mais un **espace de positions** qui évolue selon le contexte et le temps. L'app ne "révèle" pas qui on est — elle cartographie comment on se positionne.

## Angle différenciateur

Plutôt que de classer l'utilisateur dans une case (MBTI = 16 types), proposer un **profil vectoriel continu** : chaque dimension est un spectre, pas une case. Le résultat est une empreinte unique, pas une étiquette.

Deuxième différenciateur : **le résultat change**. La personnalité évolue. L'app reteste périodiquement et montre la trajectoire. On ne lit pas "tu es INFJ" — on lit "tu t'es déplacé vers plus d'extraversion depuis 3 mois".

## Questions ouvertes

- Format : questions QCM rapides (type Tinder swipe) ou scénarios longs ?
- Résultat statique ou évolutif dans le temps (la personnalité change) ?
- Interface gamifiée ou sobre ?
- Connexion avec APP-MOBILE-GRID : l'axe X = variantes de contexte (au travail / en famille) ?
