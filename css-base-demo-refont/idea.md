## Description

Refonte complète des 10 démos de `@medyll/css-base` pour exploiter les nouvelles capacités de la skill mise à jour.

**Mission :** Refaire toutes les démos avec :
- Seed tokens (pas les tokens calculés)
- CSS `@function` (Chrome 139+) — color manipulation
- `attr()` utilities (Chrome 139+) — dynamic styling
- `light-dark()` — dark mode automatique
- References `references/inspiration/`

**Démos concernées (10) :**
1. netflix — Dark-first, rouge Netflix
2. amazon — Light-first, orange + secondary
3. whatsapp — Vert WhatsApp
4. windows — Windows 11, bleu
5. macos — macOS, light-first
6. spotify — Dark, vert Spotify
7. slack — Purple Slack
8. github — GitHub light/dark
9. dashboard — Dashboard générique
10. terminal — Terminal, monospace

## Potentiel

- Démos comme documentation vivante du design system
- Showcase des capacités avancées (CSS @function, attr())
- Référence pour la communauté
- Validation du skinning system
